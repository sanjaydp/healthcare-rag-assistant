# Healthcare RAG Assistant

A domain-specific AI assistant that uses Retrieval-Augmented Generation (RAG) to answer questions over healthcare documents.

## Tech Stack
- Python
- FastAPI
- Streamlit
- LangChain
- LangGraph
- ChromaDB
- OpenAI API
